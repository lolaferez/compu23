En el código se tiene establecido el algoritmo para simular el modelo de Ising para un sistema 
Cuadrado de NxN partículas (en este caso 100).
Se adjunta la animación .py así como el .txt de las matrices para la animación.

Las pruebas realizadas son tres:

1. "Ising_raro.mp4" se parte de un sistema desordenado y a una temperatura de T=1.1 el sistema finalmente se estabiliza aunque con una banda definida y estática que divide spines -1 y 1.

2. "OrdenadoT_0.4.mp4" muestra un sistema desordenado que finalmente se estabiliza para una temperatura de T=0.4.

3. "DesordenadoT_2.8.mp4" aquí se parte de un sistema ordenado y se muestra como se desordena para una temperatura de T=2.8.
