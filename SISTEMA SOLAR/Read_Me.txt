En esta carpeta, se adjunta el código del programa que permite simular el sistema solar. [SISTEMA HELIOCENTRICO]

Como archivos .txt se usan los que contienen las masas de los planetas, las velocidades iniciales de cada uno de ellos así como las posiciones iniciales. (Para poder cambiar de manera sencilla las condiciones iniciales)

El código genera los datos de los archivos de energía total de cada planeta, así como sus periodos en días y las posiciones en el plano de cada uno en cada instante "posicionesitera.txt". Con esto último, y el .py para la animación se muestra en "SistemaSolar.mp4" el sistema solar simulado.
